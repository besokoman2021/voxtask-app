# 🎙️ منظم الحياة - منظم المهام الصوتي الذكي (PWA)

تطبيق **منظم الحياة** هو تطبيق ويب تقدمي (PWA) مصمم خصيصاً لمساعدتك على تنظيم يومك وإنجاز مهامك بذكاء وسرعة. يجمع التطبيق بين **التنبيه الصوتي باللغة العربية** وميزة **الإملاء الصوتي (إملاء المهام بصوتك)** وتتبع تفاعلي للأداء ومستوى الإنتاجية اليومي، كل هذا بتصميم زجاجي عصري وجذاب باللون الأزرق السماوي المريح للعين.

---

## ✨ المميزات الرئيسية
- 📢 **تذكير صوتي ذكي**: ينطق بوضوح عنوان وتفاصيل المهمة باللغة العربية فور حلول وقتها المبرمج.
- 🎙️ **إملاء المهام صوتياً**: اكتب مهامك بمجرد التحدث للميكروفون دون الحاجة للكتابة اليدوية.
- 📈 **لوحة تقييم الأداء**: احصل على نسبة مئوية دقيقة لإنجازاتك اليومية عبر رسم بياني دائري متفاعل تلقائياً.
- 🏆 **مستويات الإنتاجية والأيام المتتالية**: تحدى نفسك وحافظ على تقدمك اليومي (Streaks) لتنتقل من مستوى "مبتدئ" إلى "بطل الإنتاجية"!
- 📲 **دعم PWA بالكامل**: ثبّت التطبيق على شاشة هاتفك أو جهاز الكمبيوتر ليعمل بشكل مستقل وبسرعة فائقة.
- 📶 **يعمل دون إنترنت (Offline Mode)**: بفضل تقنية الـ Service Worker، يمكنك متابعة وإضافة مهامك في أي مكان حتى بدون اتصال بالشبكة.

## 🚀 كيفية تشغيل التطبيق محلياً
يمكنك تشغيل التطبيق ببساطة شديدة وبأكثر من طريقة:

### الطريقة الأولى: التشغيل المباشر
فقط اضغط نقرتين مزدوجتين على ملف `index.html` في متصفحك (يفضل Google Chrome أو Microsoft Edge لدعم تقنية الإملاء الصوتي بشكل ممتاز).

### الطريقة الثانية: تشغيل خادم محلي (موصى به لتثبيت PWA)
باستخدام خادم محلي مثل `Live Server` في VS Code أو باستخدام `python`:
```bash
# إذا كان لديك Python مثبت
python -m http.server 8000
```
ثم افتح الرابط التالي في متصفحك: `http://localhost:8000`

## 🌐 دليل رفع ونشر التطبيق على GitHub Pages (مجاناً بثوانٍ!)
لنشر التطبيق ومشاركته مع الجميع بشكل مباشر على الإنترنت، اتبع الخطوات البسيطة التالية:

### الخطوة 1: تهيئة Git في مجلد المشروع
افتح مبوبة الأوامر (Terminal) في مجلد التطبيق وقم بتنفيذ الأوامر التالية:
```bash
# 1. تهيئة المستودع المحلي
git init

# 2. إضافة جميع ملفات التطبيق
git add .

# 3. تسجيل التعديلات الأولى
git commit -m "إطلاق الإصدار الأول من تطبيق منظم الحياة الصوتي"
```

### الخطوة 2: ربط المستودع بـ GitHub
1. اذهب إلى موقع [GitHub](https://github.com) وقم بإنشاء مستودع جديد (Repository) باسم `life-organizer` مثلاً.
2. انسخ رابط المستودع الجديد ثم اكتب في الـ Terminal:
```bash
# استبدل الرابط أدناه برابط مستودعك الخاص
git remote add origin https://github.com/USERNAME/life-organizer.git

# تغيير الفرع إلى الفرع الرئيسي
git branch -M main

# رفع الملفات إلى جيت هاب
git push -u origin main
```

### الخطوة 3: تفعيل GitHub Pages لعرض التطبيق مجاناً
1. بعد رفع الملفات، اذهب إلى صفحة المستودع الخاص بك على GitHub.
2. اضغط على زر **Settings** (الإعدادات) في القائمة العلوية.
3. من القائمة الجانبية اليسرى، اضغط على **Pages**.
4. تحت قسم **Build and deployment** وفي خيار **Branch**:
   - قم بتغيير الفرع من `None` إلى `main`.
   - اترك المجلد المختار `/ (root)` كما هو.
   - اضغط على زر **Save** (حفظ).
5. انتظر دقيقة واحدة، وسيعطيك GitHub رابطاً مباشراً لموقعك مثل:
   `https://USERNAME.github.io/life-organizer/`
